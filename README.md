# Foundations

Self-contained, iframe-embeddable HTML built for the Brainnovation Network.

Open `index.html` in a browser, or embed via:

```html
<iframe src="https://BrainnovationNetwork.github.io/bn-foundations/" width="100%" height="900" frameborder="0"></iframe>
```

## Notes

- All assets (CSS, JS, fonts, logo SVG) are embedded inline so the file works without external dependencies.
- Designed for the Mighty Pro community platform but iframe-embeddable anywhere.
- This is a self-assessment or learning tool, not medical advice. Not a diagnosis.

Built with help from the Brainnovation Network.
